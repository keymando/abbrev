Abbrev
----------

Keymando plugin that replaces an mnuemonic with text.

Examples
========

``` ruby

  abbrev 'nname', 'John Smith'

  abbrev 'addr', '123 Test Street, New York, NY, 12345'
  
  abbrev 'ssmilies' do
    send(':)')
    send(':)')
    send(':)')
  end

```

Requirements
============

Keymando version 1.0.3
